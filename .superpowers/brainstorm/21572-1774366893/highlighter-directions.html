原生静态高亮，选哪种更对味？

我先按你现在这个 App 的气质做了 3 个方向，重点看“像不像手写标注”，不是看颜色本身。

A · 手写下划线
搜索书名、作者、 课程或自然语言

更像重点标注

轻、克制、适合标题或短句，不会把信息面压得太满。

B · 马克笔底色
搜索书名、作者、 课程或自然语言

更像 Magic UI 那张图

识别度最高，视觉更活，但面积会比下划线更大一点。

C · 便签感色块
搜索书名、作者、 课程或自然语言

更有纸质笔记感

风格化更强，适合首页氛围，但如果频繁使用会更抢戏。

问题只问一个：你更想让这个效果偏向哪一类？直接点选 A / B / C 就行，也可以在终端里补一句你想微调的地方。
